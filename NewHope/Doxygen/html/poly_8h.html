struct  poly
 Contains an N sized array of 16bit coefficients. /n Elements of R_q = Z_q[X] / (X^n + 1). /n Represents polynomial coeffs[0] + X * coeffs[1] + X^2 * xoeffs[2] + ... + X^{n-1} * coeffs[n-1]. More...
