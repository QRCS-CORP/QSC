\begin{DoxyRemark}{Remarks}
Classic Mc\+Eliece is a KEM designed for IND-\/\+CCA2 security at a very high security level, even against quantum computers. ~\newline
The KEM is built conservatively from a PKE designed for OW-\/\+CPA security, namely Niederreiter\textquotesingle{}s dual version of Mc\+Eliece\textquotesingle{}s PKE using binary Goppa codes. ~\newline
Every level of the construction is designed so that future cryptographic auditors can be confident in the long-\/term security of post-\/quantum public-\/key encryption. ~\newline
 Based entirely on the C reference branch of Dilithium taken from the NIST Post Quantum Competition Round 3 submission. ~\newline
The NIST Post Quantum Competition \href{https://csrc.nist.gov/Projects/post-quantum-cryptography/round-3-submissions}{\texttt{ Round 3}} Finalists. ~\newline
The \href{https://classic.mceliece.org/}{\texttt{ Mc\+Eliece}} website. ~\newline
The Mc\+Eliece \href{https://classic.mceliece.org/nist/mceliece-20201010.pdf}{\texttt{ Algorithm}} Specification. ~\newline
Authors\+: Daniel J. Bernstein, Tung Chou, Tanja Lange, and Peter Schwabe. ~\newline
Updated by John Underhill, June 28 2021. 
\end{DoxyRemark}
